# Comrade Champion League (CCL)

## Run in Antigravity

1. Open this folder in Antigravity
2. Open the terminal
3. Run: `node server.js`
4. Website opens at: `http://localhost:3000`
5. Admin Panel at: `http://localhost:3000/admin`

## Admin Password
Default: `ccl2025` (change in server.js line 15)

## Files
- `server.js` - Backend server (zero dependencies)
- `index.html` - Main tournament website
- `admin.html` - Admin panel (edit/delete teams)
- `teams.json` - Auto-created database file
- `package.json` - Project config

## No npm install needed!
Server uses only built-in Node.js modules. Zero external dependencies.
